Batches¶

Data flow is bidirectional in gunpowder: A BatchRequest is sent upstream (i.e., from sink to source) and a Batch is sent downstream to fulfill the request.

class gunpowder.BatchRequest(array_specs=None, points_specs=None)[source]¶

A collection of (possibly partial) ArraySpec and PointsSpec forming a request.

For usage, see the documentation of ProviderSpec.

class gunpowder.Batch[source]¶

Contains data that is passed through the pipeline from sources to sinks.

arrays¶

dict from ArrayKey to Array – Contains all arrays that have been requested for this batch.

points¶

dict from PointsKey to Points – Contains all point sets that have been requested for this batch.
